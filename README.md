Customer Sales Analysis

Project Overview

This project analyzes customer sales data using Python, Pandas, and Matplotlib. It processes raw data, computes key metrics, and presents insights through visualizations.

Tools and Technologies

- Python
- Pandas
- Matplotlib

Dataset

The dataset includes:

- Customer
- City
- Product
- Category
- Price
- Quantity

Features

- Total sales calculation (Price × Quantity)
- City-wise sales analysis
- Top 3 products identification
- Top 3 customers identification
- Category-wise distribution

Dashboard

The project generates:

- Bar chart for city-wise sales
- Bar chart for top products
- Bar chart for top customers
- Pie chart for category distribution

Key Insights

- Identifies the highest revenue-generating city
- Determines top-performing products
- Highlights high-value customers
- Shows category contribution in percentage

How to Run

1. Install dependencies:
   pip install pandas matplotlib

2. Run the script:
   python analysis.py
